 Let $a_{i}$ be the hay cow $i$ likes, if there exists adjacent hays such that  $a_{i} = a_{i+1}$,we can run a focus group on $i, i+1, i+2$ or $i-1, i, i+1$. This will result in cow $i$ and $i+1$ infecting cow $i+2$ or $i-1$ with the type of hay they like. We can perpetually run focus groups like this to eventually infect every cow with hay $a_{i}$. 
 
 Occurences of $a_{i} = a_{i+2}$ will also create adjacent hays of the same type (by running a focus group on $i,i+1, i+2$) and therefore this  also makes $a_{i}$ part of the answer.
 


**Time Complexity:** $\mathcal{O}(N)$

```cpp
#include <bits/stdc++.h>
using namespace std;

void solve() {
    int n;
    cin >> n;
    vector<int> hay(n);
    for(int i = 0; i < n; i++) cin >> hay[i];

    set<int> results;
    for(int i = 0; i < n-1; i++) {
        if(i + 1 < n && hay[i] == hay[i + 1]) results.insert(hay[i]);

        if(i + 2 < n && hay[i] == hay[i + 2]) results.insert(hay[i]);
    }

    if(results.empty()) {
        cout << -1;
    }
    else {
		// Printing out the first element separately to be conjugated with a space to avoid an extra space
        cout << *results.begin(); results.erase(results.begin()); 
        for(int i : results) cout << " " << i;
    }
    
    cout << endl;
}

int main() {
    int t;
    cin >> t;
    while(t--) {
        solve();
    }
}
```
